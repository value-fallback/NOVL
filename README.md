## Installation

**Via [NPM](https://www.npmjs.com):**

[![NPM](https://nodei.co/npm/novl.png?downloads=true&downloadRank=true&stars=true)](https://www.npmjs.com/package/novl)

```
npm install novl
```

**Via [Yarn](https://yarnpkg.com):**

```
yarn add novl
```
